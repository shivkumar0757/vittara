# Vittara

Personal finance OS built on Maybe Finance.

## Setup

**To self-host Vittara, see the [Docker hosting guide](docs/hosting/docker.md).**

The instructions below are for local development.

### Requirements

- See `.ruby-version` file for required Ruby version
- PostgreSQL >9.3 (ideally, latest stable version)

After cloning the repo:

```sh
cd vittara
cp .env.local.example .env.local
bin/setup
bin/dev

# Optionally, load demo data
rake demo_data:default
```

Visit http://localhost:3000. Default credentials (generated by DB seed):

- Email: `user@maybe.local`
- Password: `password`
